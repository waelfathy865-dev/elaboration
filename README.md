## 📋 نظرة عامة

منصة نبض المدارس هي نظام متكامل لإدارة المدارس يربط مديري المدارس بالإدارة العامة من خلال نظام موحد واحد حيث يمكن لكل مدرسة:

- 📝 تقديم التقارير والمشاكل
- 💡 تقديم المقترحات والآراء
- 📊 رفع الإحصائيات اليومية
- 💬 التواصل المباشر مع الإدارة
- 🔔 تلقي التنبيهات الفورية

## 🎨 المواصفات

- **الواجهة**: لوحة تحكم حكومية حديثة وعصرية
- **التصميم**: Glassmorphism + Material Design
- **الألوان**: أزرق، أبيض، أخضر (حكومي)
- **اللغات**: العربية (RTL) + الإنجليزية (LTR)
- **الاستجابة**: سطح المكتب، التابلت، الهاتف الذكي

## 👥 أدوار المستخدمين

1. **Super Admin** - المسؤول الأساسي
2. **General Director** - المدير العام
3. **Administration Managers** - مديرو الإدارات
4. **School Principal** - مدير المدرسة
5. **Vice Principal** - مساعد مدير المدرسة (اختياري)
